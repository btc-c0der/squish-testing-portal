Quick Start

Your First Squish Test
import squish
import test

def test_basic_interaction():
    # Start your application
    startApplication("myapp")
    
    # Find and interact with objects
    button = findObject(":OK_QPushButton")
    clickButton(button)
    
    # Verify the result
    test.verify(findObject(":Result_QLabel").visible)
    
    # Clean up
    closeApplication()

This simple example demonstrates the basic structure of a Squish test. Explore more examples to learn advanced techniques.
